Customizing the Starterr theme

The Starterr theme has been built with the support of Node, Bower and Grunt.

Quick start

To edit its source files just open a terminal window at the theme root folder, and type:

npm i && bower i

Then just start watching:

grunt watch

Credits

Sass

Bourbon and Scut together may seems an overload but since they are mixing libraries you'll generate just from what you'll use ; )

  • Bourbon by thoughtbot A simple and lightweight mixin library for Sass
  • Scut by David Clark Sass utilities for the frontend laborer
  • Jeet by MojoTech The most advanced, yet intuitive, grid system available for Sass or Stylus
  • Mq by the guys at The Guardian A Sass mixin that helps manipulating media queries in an elegant way
  • Knife by Paul van Zyl Nail vertical rhythm, modular scale, and REMs like a boss with this simple set of SASS/SCSS variables, functions and mixins.

Javascript

All the libs here are standalone pure javascript libraries, for maximum performance on mobile and full support for desktop browsers.

  • textFit by Samuel Reed A jQuery-free component that quickly fits single and multi-line text to the width (and optionally height) of its container.
  • FluidVids by Todd Motto A lightweight, easy-to-use jQuery plugin for fluid width video embeds.
  • iDangerous Swiper by iDangero.us Mobile touch slider & framework with hardware accelerated transitions
  • FastClick by the guys at FTLabs Polyfill to remove click delays on browsers with touch UIs
